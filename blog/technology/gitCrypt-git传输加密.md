- 环境：Fedora29 64bit ，mini安装，中文环境。

```
# yum install git -y
# yum install gcc -y
# yum install gcc-c++ -y
# yum install openssl-devel -y
# yum install make -y  #这个命令非必须。
# cd /opt
# git clone https://www.agwa.name/git/git-crypt.git
# make
# make install

```
